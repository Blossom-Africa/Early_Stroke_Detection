# Early_Detection_Stroke
## Problem Statement
Stroke is a leading cause of death and long-term disability worldwide, often linked to preventable health and lifestyle factors. Early identification of at-risk individuals is crucial, but the roles of factors like age, hypertension, heart disease, glucose levels, and smoking are not fully understood. 

## Objective
- This project seeks to uncover key predictors and support timely intervention and preventive healthcare decisions
